# 6-DOF Robotic Arm

An independent robotics project: a 6-degree-of-freedom robotic arm built from scratch, controlled via Arduino.

## 🔧 Status: In Progress
- ✅ Base and shoulder joint designed (Fusion 360) and fabricated
- ✅ 6x MG996R servo motors wired and integrated
- ✅ Basic servo control programmed via Arduino IDE
- 🔲 Gripper design and fabrication — in progress
- 🔲 Final mechanical assembly and calibration
- 🔲 Full demo video

## Overview
This project explores end-to-end robotic arm development — mechanical design, CAD modelling, servo integration, and embedded control — as a self-driven way to build practical robotics skills alongside coursework.

## Hardware
- 6x MG996R servo motors
- Arduino (Uno)
- Custom-fabricated base and joint assembly

## Software
- CAD: Autodesk Fusion 360
- Firmware: Arduino IDE (C/C++)
